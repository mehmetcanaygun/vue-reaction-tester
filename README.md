## Reaction Tester

A small game which you try to click on shapes as many as you can in a little time. Built with Vue 3, TS and TailwindCSS.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
